# dark mode demo

a simple, modern portfolio-style website with a pure dark theme.

## features

- **pure CSS** — no frameworks, no external dependencies
- **responsive** — optimized for mobile, tablet, and desktop
- **gradients** — elegant purple-blue accent colors
- **clean design** — minimalist, focus on content

## getting started

just open the `index.html` file in your browser — no build step required.

## customization

edit the styles in the `<style>` block to change colors, fonts, or layout.
